第一章：极限与连续

📚 本章导读

极限是微积分的基础概念，它描述了当自变量趋近某个值时，函数值的变化趋势。本章将学习：

  • 数列极限的定义与性质
  • 函数极限的概念与计算方法
  • 无穷小与无穷大的比较
  • 函数的连续性与间断点
学习建议：极限概念较为抽象，建议结合图形理解，多做练习题加深印象。

定义 1.1：数列极限

设 {an} 是一个数列，如果存在常数 A，使得对于任意给定的正数 ε，总存在正整数 N，当 n > N 时，恒有：

\[|a_n - A| < \varepsilon\]

则称常数 A 是数列 {an} 的极限，记作：

\[\lim_{n \to \infty} a_n = A\]

定义 1.2：函数极限

设函数 f(x) 在点 x0 的某一去心邻域内有定义，如果存在常数 A，使得对于任意给定的正数 ε，总存在正数 δ，当 0 < |x - x0| < δ 时，恒有：

\[|f(x) - A| < \varepsilon\]

则称常数 A 是函数 f(x) 当 x 趋向于 x0 时的极限。

夹逼定理

如果数列 {xn}, {yn}, {zn} 满足：

  1. 从某项起，恒有 yn ≤ xn ≤ zn
  2. lim yn = lim zn = a

则 lim xn = a

两个重要极限

\[\lim_{x \to 0} \frac{\sin x}{x} = 1\]
\[\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e\]

例 1：计算极限

题目：计算 \(\lim_{x \to 0} \frac{\tan x - \sin x}{x^3}\)

解：

首先，我们将 tan x 和 sin x 展开：

\[\tan x - \sin x = \sin x \left(\frac{1}{\cos x} - 1\right) = \sin x \cdot \frac{1 - \cos x}{\cos x}\]

利用等价无穷小替换：当 x → 0 时：

  • sin x ~ x
  • 1 - cos x ~ x²/2
  • cos x → 1

因此：

\[\lim_{x \to 0} \frac{\tan x - \sin x}{x^3} = \lim_{x \to 0} \frac{x \cdot \frac{x^2/2}{1}}{x^3} = \frac{1}{2}\]

例 2：证明数列收敛

题目：证明数列 \(a_n = \frac{n^2 + 1}{n^2 + n + 1}\) 收敛并求其极限。

证明：

将分子分母同时除以 n²：

\[a_n = \frac{1 + \frac{1}{n^2}}{1 + \frac{1}{n} + \frac{1}{n^2}}\]

当 n → ∞ 时：

\[\lim_{n \to \infty} a_n = \frac{1 + 0}{1 + 0 + 0} = 1\]

因此数列收敛，极限为 1。
